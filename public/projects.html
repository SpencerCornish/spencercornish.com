Projects

These are some projects I've worked on, both in and out of school. There are even more available to peruse on my personal Github, located here

MSUBot A course registration notifier for Montana State University
MSUBot is an application which scrapes the number of open seats for classees at MSU. If a class is full, this allows students to be notified via text message when a seat opens up. The application was written using Dart for the frontend, and Go for the backend.

Unwent.com A angular web app that rings a robo-gong
Unwent.com is a web app written in Angular that rings a gong. The gong is a robotic arm attached to, well, a gong.

Cat's Conundrum car electronics A Custom PCB for Our Remote Control Car
A two layer PCB, designed in Autodesk Eagle. This was used to facilitate easy and clean electronics for our remote controlled car project in Interdisciplinary Engineering.

Cat's Conundrum car control app A Custom Flutter App
An app, written in flutter, to control our car for EGEN 310. This was my first flutter project, and additionally the first time I utilized a Redis Database.
